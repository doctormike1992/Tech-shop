# My E-Shop Project


## Description
This is a **front-end E-Shop project** built with **HTML, CSS, JavaScript, and React**.  
It allows users to browse products, add them to a cart, mark favorites, and manage orders. The project uses **Firebase** for authentication, storage, and database management.

---

## Features
- Browse products by categories (Headphones, Laptops, Phones, Accessories, etc.)
- Product detail pages with description, specifications, and price
- User authentication (sign up, login, logout)
- Add products to **cart** and manage orders
- Add products to **favorites**
- Admin panel to manage products and view all user orders
- Responsive design for desktop and mobile
- Smooth animations and interactive UI elements

---

## Tech Stack
- **Frontend:** React, TailwindCSS, JavaScript, HTML, CSS  
- **Backend / Database:** Firebase Authentication, Firestore, Firebase Storage  
- **Routing:** React Router  
- **State Management:** Redux Toolkit
